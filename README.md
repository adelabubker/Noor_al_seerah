# 🌙 نور السيرة — Noor Al-Seerah

> موقع تعليمي إسلامي يُقدِّم سيرة النبي ﷺ وسير الصحابة الكرام والغزوات

---

## 📋 نظرة عامة

**نور السيرة** تطبيق ويب من صفحة واحدة (SPA) مبني بـ React + Vite، يهدف إلى تقديم المحتوى الإسلامي التعليمي بطريقة تفاعلية وجميلة. يشمل المشروع:

- **سيرة النبي ﷺ** عبر خط زمني تفاعلي
- **سير الصحابة الكرام** مع تفاصيل شاملة وأحاديث
- **الغزوات النبوية** مع إمكانية البحث والتصفية
- **لوحة تحكم إدارية** لإدارة المحتوى

---

## 🛠️ التقنيات المستخدمة

| التقنية | الوصف |
|---------|-------|
| **React** | مكتبة واجهة المستخدم |
| **Vite** | أداة البناء والتطوير |
| **React Router** | التنقل بين الصفحات |
| **Tailwind CSS** | تنسيق الواجهة |
| **PostCSS + Autoprefixer** | معالجة CSS |
| **localStorage** | تخزين بيانات الغزوات محلياً |

---

## 🚀 تشغيل المشروع

### المتطلبات
- Node.js (الإصدار 18 أو أحدث)
- npm أو yarn

### خطوات التثبيت

```bash
# 1. استنساخ المشروع
git clone https://github.com/your-username/noor-al-seerah.git
cd noor-al-seerah

# 2. تثبيت التبعيات
npm install

# 3. تشغيل بيئة التطوير
npm run dev

# 4. البناء للإنتاج
npm run build

# 5. معاينة النسخة المبنية
npm run preview
```

## ✨ الميزات الرئيسية

### 🕌 السيرة النبوية
- خط زمني أفقي تفاعلي على الشاشات الكبيرة
- خط زمني عمودي على الشاشات الصغيرة
- لوحة تفاصيل لكل حدث مع التنقل بين الأحداث

### 👥 الصحابة الكرام
- بحث نصي حر مع تصفية بالتصنيف
- صفحة تفاصيل لكل صحابي تشمل: السيرة، الإنجازات، الأحاديث
- التنقل بواسطة الـ slug

### ⚔️ الغزوات
- بيانات محفوظة في `localStorage` (قابلة للتعديل)
- فلترة حسب المرحلة، النتيجة، ونص البحث
- عرض فيديو YouTube مدمج
- نافذة تفاصيل كاملة لكل غزوة

### 🛡️ لوحة التحكم الإدارية
- إدارة الصحابة (عرض فقط في النسخة الحالية)
- إدارة الغزوات بالكامل (إضافة، تعديل، حذف)
- إدارة المستخدمين (بيانات placeholder)

## 🎨 نظام الثيمات

يدعم الموقع الوضع الفاتح والداكن:
- يتم حفظ اختيار المستخدم في `localStorage`
- الإعدادات المخصصة في `tailwind.config.js`
- متغيرات CSS عامة في `src/index.css`

---

## ⚙️ ملاحظات تقنية

- **البيانات الثابتة**: بيانات الصحابة والسيرة ثابتة وللقراءة فقط.
- **البيانات المتغيرة**: الغزوات تُحفظ في `localStorage` وتدعم CRUD كامل.
- **لا يوجد backend حقيقي**: طبقة API في `services.js` محاكاة فقط. عمليات الكتابة على الصحابة ترمي خطأ مقصوداً.
- **المصادقة وهمية**: حساب المدير مشفر مباشرة في `AuthContext.jsx`.

---

## 🌐 دعم اللغة العربية

- خطوط وألوان عربية مخصصة عبر Tailwind
- تصميم كامل يراعي القراءة من اليمين لليسار
